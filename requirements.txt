# Hybrid LLM: free local Ollama + cost-tiered Anthropic Claude
# ============================================================================
# The pipeline is hybrid. Free local Ollama (HTTP via the already-present
# `requests` package) handles tasks it does as well; the official Anthropic SDK
# handles the complex agentic reasoning (differential dx, cure-research) on a
# cost-tiered basis (Haiku/Sonnet/Opus — you control the spend in .env).
anthropic>=0.69.0
# Ollama is a native app (not a pip package). Install + pull a small edge model:
#     winget install Ollama.Ollama   # or https://ollama.com/download
#     ollama pull llama3.2:3b
# Set ANTHROPIC_API_KEY to enable the Claude tier; without it the system runs in
# reduced (Ollama/deterministic) form. See scripts/setup_ollama.*.
#
# The effectiveness PDF report is rendered with Pillow (already listed above),
# avoiding matplotlib/reportlab which lack reliable Python 3.14 win-arm64 wheels.
